# 📖 Project Overview

Store Ratings is a full-stack web application that allows users to browse registered stores, submit ratings between **1 and 5**, and access role-based dashboards. The application follows secure authentication practices and implements role-based authorization for Administrators, Store Owners, and Normal Users.

---

# ✨ Features

## 👨‍💼 System Administrator

- Dashboard
- Manage Users
- Manage Stores
- View Ratings
- Search & Filter
- Sorting
- Logout

## 👤 Normal User

- Register
- Login
- Browse Stores
- Search Stores
- Submit Ratings
- Update Ratings
- Change Password
- Logout

## 🏪 Store Owner

- Login
- View Average Rating
- View Users who Rated Store
- Update Password
- Logout
